# 🚁 F450无人机IMU教学演示系统

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](你的Streamlit链接)

面向高职《无人机传感器与检测技术》课程的交互式教学平台。

## 🎯 功能特性
- **3D姿态可视化**: F450四旋翼实时姿态显示
- **陀螺仪校准**: 零偏采集与补偿算法演示
- **加速度计校准**: 水平校准与六面法精密校准
- **误差模拟**: 可调节噪声、漂移，观察对导航的影响

## 🚀 快速开始

### 本地运行
```bash
pip install -r requirements.txt
